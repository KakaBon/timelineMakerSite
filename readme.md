# 网页版时间轴制作工具

## 简介

这是一个基于 Vite 和 vis-timeline 的网页版时间轴制作工具。

项目包含以下主要功能：

- 通过导入csv文件将事件呈现在时间轴上
- 导入和导出 CSV 文件
- csv数据内容的增删改查及时间轴上事件信息的实时更新
- 数据文本查找替换和事件查找功能
- 缩放和拖拽时间轴
- 事件类别的泳道显示和颜色区分
- 事件上鼠标悬浮查看更多信息
